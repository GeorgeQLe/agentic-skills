# Creator Foundation Pack

Project-local workflows for creator-media research foundations, cross-platform evidence, creator dossiers, positioning, programming, series planning, product-led media mapping, and metrics review.

Default flow:

```text
research-bootstrap -> research-directory-conventions
-> creator-platform-capability-matrix -> creator-evidence-schema
-> creator-presence-dossier -> creator-positioning -> content-programming
-> series-spec -> product-led-media-map -> creator-metrics-review
```

Skills:

```text
research-bootstrap, research-directory-conventions,
creator-platform-capability-matrix, creator-evidence-schema,
creator-presence-dossier, creator-positioning, content-programming,
series-spec, product-led-media-map, creator-metrics-review
```

Install `youtube-ops` for YouTube-specific audit and optimization work. Install `remotion` for format research, scripts, and Remotion builds.
