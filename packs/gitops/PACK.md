# GitOps Pack

Git operations skills for syncing and feature-based commit workflows. Provides structured approaches to git synchronization and organized commit strategies.

Install this pack when needing git sync or structured commit workflows.

Install in a project with:

```bash
scripts/pack.sh install gitops
```

Then refresh local skill links if needed:

```bash
scripts/pack.sh refresh
```

## Skills

- `sync`: Synchronize branches, remotes, and working trees with conflict resolution.
- `commit-and-push-by-feature`: Organize and commit changes grouped by feature with structured commit messages.
