# Code Debug Pack

Debugging and investigation skills for diagnosing and resolving issues. Covers structured investigation, interactive debugging, and request flow tracing.

Install this pack when investigating bugs, debugging issues, or tracing request flows through a system.

Install in a project with:

```bash
scripts/pack.sh install code-debug
```

Then refresh local skill links if needed:

```bash
scripts/pack.sh refresh
```

## Skills

- `investigate`: Systematically investigate a reported issue by gathering evidence and narrowing root causes.
- `debug`: Interactively debug a specific issue with breakpoints, logging, and hypothesis testing.
- `trace`: Trace a request or data flow through the system to identify where behavior diverges from expectations.
