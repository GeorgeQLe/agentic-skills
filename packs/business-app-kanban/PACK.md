# Business App Kanban Pack

Opt-in PoketoWork kanban workflow variants for business-app projects.

Install this pack only when the project intentionally uses PoketoWork kanban. It does not install the base `business-app` pack.

**Primary role:** Both — inherits `business-app` (Claude-orchestration) and adds kanban `run`/`ship`/`ship-end` execution variants. See `docs/operating-modes.md` § "Pack emphasis".
