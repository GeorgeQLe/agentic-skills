# Docs Health Pack

Documentation health and project hygiene auditing skills. Covers project structure auditing, convention violation detection, and developer documentation reconciliation.

Install this pack when auditing project structure, convention violations, or reconciling dev docs with actual implementation.

Install in a project with:

```bash
scripts/pack.sh install docs-health
```

Then refresh local skill links if needed:

```bash
scripts/pack.sh refresh
```

## Skills

- `hygiene`: Audit project structure and conventions for violations, stale artifacts, and organizational issues.
- `reconcile-dev-docs`: Compare developer documentation against the actual codebase and flag discrepancies.
