# Session Analytics Pack

Session analysis and triage skills for understanding agent session history. Covers session log analysis and incident triage.

Install this pack when analyzing Claude or Codex session history or triaging session incidents.

Install in a project with:

```bash
scripts/pack.sh install session-analytics
```

Then refresh local skill links if needed:

```bash
scripts/pack.sh refresh
```

## Skills

- `analyze-sessions`: Analyze agent session logs for patterns, efficiency, and improvement opportunities.
- `session-triage`: Triage session incidents to identify root causes and recommend fixes.
