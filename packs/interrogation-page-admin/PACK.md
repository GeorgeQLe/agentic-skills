# Interrogation Page Admin Pack

Interrogation page administration skills for auditing and upgrading stage-zero interrogation round pages to the current local interrogation-page standard.

Install this pack when auditing or upgrading generated `interrogation/*.html` round pages for a project.

Install in a project with:

```bash
scripts/pack.sh install interrogation-page-admin
```

Then refresh local skill links if needed:

```bash
scripts/pack.sh refresh
```

## Skills

- `upgrade-interrogation-pages`: Audit and explicitly upgrade generated `interrogation/*.html` round pages to the current local interrogation-page standard.
