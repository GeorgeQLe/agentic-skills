# Teardown Pack

Project teardown and decommission skills for cleanly winding down projects or components. Covers both rapid teardown and structured decommissioning.

Install this pack when tearing down or decommissioning a project, service, or component.

Install in a project with:

```bash
scripts/pack.sh install teardown
```

Then refresh local skill links if needed:

```bash
scripts/pack.sh refresh
```

## Skills

- `desk-flip`: Rapid project teardown when a clean break is needed.
- `decommission`: Structured decommissioning with dependency checks, data migration planning, and cleanup verification.
