# Business Growth Pack

Project-local workflows for GTM, growth, pricing, metrics, experiments, landing copy, and PMF validation.

Install this pack when a project has enough discovery and lifecycle evidence to plan engagement loops, define success metrics, design pricing, build go-to-market strategy, write landing copy, run validation experiments, model growth loops, or measure product-market fit.

Default flow:

```text
hook-model -> metrics -> monetization -> gtm -> landing-copy -> experiment -> growth-model -> pmf-assessment
```

Skills:

```text
hook-model, growth-model, metrics, gtm, monetization,
landing-copy, pmf-assessment, experiment
```

Install `business-research` when ICP, positioning, or market evidence is missing. Install `customer-lifecycle` when journey, onboarding, conversion, transaction, retention, or expansion evidence is missing. Install `business-ops` for post-launch reviews, runway, risks, and stakeholder reporting.
