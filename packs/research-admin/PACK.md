# Research Admin Pack

Research and documentation health management skills. Provides structured research roadmap tracking for managing documentation queues and priorities.

Install this pack when managing research documentation queues and priorities.

Install in a project with:

```bash
scripts/pack.sh install research-admin
```

Then refresh local skill links if needed:

```bash
scripts/pack.sh refresh
```

## Skills

- `research-roadmap`: Manage a structured research documentation queue with priorities, status tracking, and next-action recommendations.
