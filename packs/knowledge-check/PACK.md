# Knowledge Check Pack

Knowledge check and quiz skills for testing understanding of a codebase, domain, or concept. Provides interactive quiz-style reviews.

Install this pack when testing knowledge or running quiz-style reviews of technical content.

Install in a project with:

```bash
scripts/pack.sh install knowledge-check
```

Then refresh local skill links if needed:

```bash
scripts/pack.sh refresh
```

## Skills

- `quiz-me`: Run an interactive quiz session to test understanding of a codebase, API, or technical domain.
