# Guided Walkthrough Pack

Guided walkthrough skills for producing human-facing step-by-step instructions. Covers general guides and UAT-specific walkthroughs.

Install this pack when creating step-by-step guides or UAT walkthroughs for end users.

Install in a project with:

```bash
scripts/pack.sh install guided-walkthrough
```

Then refresh local skill links if needed:

```bash
scripts/pack.sh refresh
```

## Skills

- `guide`: Create a step-by-step guided walkthrough for a feature or workflow.
- `uat-guide`: Create a UAT-specific guided walkthrough with acceptance criteria and validation steps.
