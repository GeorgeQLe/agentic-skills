# Code Maintenance Pack

Code maintenance skills for migrations and dependency updates. Covers structured migration execution and package update workflows.

Install this pack when running migrations or updating packages and dependencies.

Install in a project with:

```bash
scripts/pack.sh install code-maintenance
```

Then refresh local skill links if needed:

```bash
scripts/pack.sh refresh
```

## Skills

- `migrate`: Execute structured code migrations with validation and rollback planning.
- `update-packages`: Update project dependencies with compatibility checks and breaking change detection.
