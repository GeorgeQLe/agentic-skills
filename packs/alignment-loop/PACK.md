# Alignment Loop

Lightweight operator-agent calibration workflow. An alternative on-ramp to the spec-interview pipeline that optimizes for shared taste, intent, and understanding before committing to full specs.

## Flow

```
taste-calibration → destination-doc → vertical-slice-splitter → implementation
```

## When to Use

- You have a rough idea and want to shape it quickly
- You value shared taste and intent over exhaustive documentation
- You want issue cards fast without a full spec-interview cycle

## When NOT to Use

- You need decision-complete specifications (`/spec-interview`)
- Regulatory or compliance documentation is required

**Primary role:** Lightweight operator-agent alignment
