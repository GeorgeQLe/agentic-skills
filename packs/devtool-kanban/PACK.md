# Devtool Kanban Pack

Opt-in PoketoWork kanban workflow variants for developer-tool projects.

Install this pack only when the project intentionally uses PoketoWork kanban. It does not install the base `devtool` pack.

**Primary role:** Both — inherits `devtool` (Claude-orchestration) and adds kanban `run`/`ship`/`ship-end` execution variants. See `docs/operating-modes.md` § "Pack emphasis".
