# Agentic Skills Bench Pack

Project-local workflows for verifying and benchmark-testing skills in this repository with the agentic-skills test harness.

Install this pack when working on `tools/agentic-skills` skill quality, benchmark reports, harness fixtures, or regression measurements. It is not for benchmarking an app, website, or product surface; the target argument is always a skill name.

**Primary role:** Skill quality measurement — run verify first, then repeated benchmark runs with pass-rate, latency, cost, and consistency reporting.
