# Website Polish Pack

Website polish skills for icon and asset management. Provides tooling for handling icons, favicons, and web asset pipelines.

Install this pack when handling icons or web assets for a website project.

Install in a project with:

```bash
scripts/pack.sh install website-polish
```

Then refresh local skill links if needed:

```bash
scripts/pack.sh refresh
```

## Skills

- `icon-handler`: Manage icon assets including favicon generation, SVG optimization, and icon set configuration.
