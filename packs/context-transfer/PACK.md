# Context Transfer Pack

Context transfer skills for session handoffs between agents or sessions. Ensures continuity when transitioning work across boundaries.

Install this pack when handing off context between sessions or agents.

Install in a project with:

```bash
scripts/pack.sh install context-transfer
```

Then refresh local skill links if needed:

```bash
scripts/pack.sh refresh
```

## Skills

- `handoff`: Package current session context, progress, and next steps into a structured handoff document for the receiving agent or session.
