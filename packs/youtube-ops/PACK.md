# YouTube Ops Pack

Project-local workflows for YouTube channel and video audits, external-video context, concept research, competitive research, packaging, metadata, portfolio, peer benchmarking, search positioning, and cadence diagnosis.

Default flow:

```text
youtube-audit -> youtube-channel-audit / youtube-video-audit
-> youtube-vid-research / youtube-concept-research / youtube-competitive-research
-> youtube-title-thumbnail-audit -> youtube-description-optimizer
-> youtube-portfolio -> youtube-peer-benchmark
-> youtube-search-positioning / youtube-cadence-diagnosis
```

Skills:

```text
youtube (entry point — intent router & play composer),
youtube-audit, youtube-channel-audit, youtube-video-audit,
youtube-vid-research, youtube-concept-research, youtube-competitive-research,
youtube-title-thumbnail-audit, youtube-description-optimizer,
youtube-portfolio, youtube-peer-benchmark, youtube-search-positioning,
youtube-cadence-diagnosis
```

Start with `/youtube` (or `$youtube` in Codex) to get routed to the right skill or compose a multi-step play.

Install `creator-foundation` when the work needs cross-platform evidence, creator positioning, programming, series, or product-led media strategy. Install `remotion` when the work needs format research, scripts, or Remotion build planning.
