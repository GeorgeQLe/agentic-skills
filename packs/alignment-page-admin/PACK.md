# Alignment Page Admin Pack

Alignment page administration skills for compiling and managing central alignment pages that consolidate project decisions, principles, and conventions.

Install this pack when compiling or managing central alignment pages for a project.

Install in a project with:

```bash
scripts/pack.sh install alignment-page-admin
```

Then refresh local skill links if needed:

```bash
scripts/pack.sh refresh
```

## Skills

- `compile-central-alignment`: Compile scattered alignment decisions, principles, and conventions into a central alignment page.
