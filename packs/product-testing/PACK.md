# Product Testing Pack

Product testing and user acceptance skills for validating features from the end-user perspective. Covers UAT journey creation, visible UAT execution, and dogfooding workflows.

Install this pack when creating UAT journeys, executing visible local UAT, or dogfooding features.

Install in a project with:

```bash
scripts/pack.sh install product-testing
```

Then refresh local skill links if needed:

```bash
scripts/pack.sh refresh
```

## Skills

- `uat`: Create human-run user acceptance testing journeys with step-by-step validation criteria.
- `visible-uat`: Execute deterministic visible UAT against local desktop, web, and app surfaces with Computer Use or the available visible UI tool.
- `dogfood`: Run structured dogfooding sessions to evaluate features from a real user perspective.
