# Product Testing Pack

Product testing and user acceptance skills for validating features from the end-user perspective. Covers UAT journey creation and dogfooding workflows.

Install this pack when creating UAT journeys or dogfooding features.

Install in a project with:

```bash
scripts/pack.sh install product-testing
```

Then refresh local skill links if needed:

```bash
scripts/pack.sh refresh
```

## Skills

- `uat`: Create and execute user acceptance testing journeys with step-by-step validation criteria.
- `dogfood`: Run structured dogfooding sessions to evaluate features from a real user perspective.
