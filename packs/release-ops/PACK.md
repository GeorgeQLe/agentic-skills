# Release Ops Pack

Release and deployment workflow skills for managing the full release lifecycle. Covers branch management, deployment execution, and release creation.

Install this pack when managing branches, deploying applications, or creating releases.

Install in a project with:

```bash
scripts/pack.sh install release-ops
```

Then refresh local skill links if needed:

```bash
scripts/pack.sh refresh
```

## Skills

- `branch-lifecycle`: Manage branch creation, merging, and cleanup through the development lifecycle.
- `deploy`: Execute deployment workflows for staging and production environments.
- `release`: Create versioned releases with changelogs and tags.
