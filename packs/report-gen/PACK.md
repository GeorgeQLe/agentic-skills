# Report Gen Pack

Report generation skills for building HTML report websites. Produces structured, visual report outputs from project data.

Install this pack when generating HTML report websites from project or analysis data.

Install in a project with:

```bash
scripts/pack.sh install report-gen
```

Then refresh local skill links if needed:

```bash
scripts/pack.sh refresh
```

## Skills

- `report-website`: Generate a self-contained HTML report website from structured data or analysis results.
