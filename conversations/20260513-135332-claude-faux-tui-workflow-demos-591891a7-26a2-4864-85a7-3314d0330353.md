## Context

The current `/workflows` page shows 8 workflow demos as animated flow-chart step sequences with a sidebar selector and metadata panels. ### Data extraction
Move workflow types and data from `src/showcase/workflows.tsx` into `src/showcase/tui/workflow-data.ts`. The existing workflows.tsx imports from it to stay backward-compatible.

### Shared hooks (`src/showcase/tui/shared/`)

- **`useTypewriter.ts`** -- character-by-character text reveal, resets on text change
- **`VariationNav.tsx`** -- horizontal strip linking v1..v5 with labels, highlights current route

```

  v3/page.tsx           -- Retro CRT

  v5/page.tsx           -- Professional Dashboard
```

### CSS strategy
Each variation gets its own CSS file scoped under a root class (`.tui-hacker`, `.tui-clean`, etc.) to prevent bleed. Import in the corresponding client component. No modifications to `globals.css`.
