Here's what I built:

**`scripts/save-conversation.sh`** ‚Äî new helper script that:
- Finds the Claude Code project directory from the repo path (uses the `-path-segments` naming convention)
- Picks the most recently modified `.jsonl` conversation file (or a specific `--session-id`)
- Converts it to readable markdown: metadata table, timestamped user/assistant turns, tool call summaries in collapsible `<details>` blocks
- Writes to `conversations/YYYYMMDD-HHMMSS-<agent>-<title-slug>.md`
- Accepts `--topic` to override the filename slug
- Gracefully fails with a warning if no conversation history is available (so it won't block Codex or headless runs)

Kanban variants (`ship-kanban`, `ship-end-kanban`) delegate to base ship with `$ARGUMENTS` passthrough, so the flag flows through automatically.

Usage: `/ship --save-conversation` or `/ship-end --save-conversation`. The conversation log gets committed and pushed to GitHub alongside the shipped work.
