New Convention Rule

Slides must fit the viewport.

Compress the slide

Use short labels, cards, chips, and matrices instead of prose blocks.

Link the detail

Put dense rationale in source artifacts and attach reference chips to the relevant slide.

Verify fit

Check desktop and mobile-sized viewports for overlap, clipping, and slide-body scrolling.

Split when needed

If content cannot fit, make another slide or link the detail out.
