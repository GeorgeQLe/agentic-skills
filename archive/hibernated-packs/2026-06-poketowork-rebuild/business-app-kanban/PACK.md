# Business App Kanban Pack

Opt-in PoketoWork kanban workflow variants for business-app projects.

Install this pack only when the project intentionally uses PoketoWork kanban. It does not install any base business domain pack; pair it with the narrow business lane that matches the current work (`business-discovery`, `business-growth`, or `business-ops`).

**Primary role:** Both — extends the business workflow with kanban `run`/`ship`/`ship-end` execution variants. See `docs/operating-modes.md` § "Pack emphasis".
