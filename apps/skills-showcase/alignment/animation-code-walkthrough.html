Open Open Phase — 8 Steps

The open phase is largely event-driven: once the drawer mounts, concurrent reactions fire. Only the in-chain boundaries (gate calls) truly freeze in stepped mode.

function completeTear() {
  hasTriggered.current = true;
  pendingOpen.current = !!autoOpenOnTear;
  debugReport({ machine: { pack: { pendingOpen: pendingOpen.current } } });
  if (autoOpenOnTear && !dbg.isStepping()) {
    pendingOpenTimer.current = setTimeout(() => {
      if (pendingOpen.current) {
        pendingOpen.current = false;
        debugReport({ machine: { pack: { pendingOpen: false } } });
        onOpen(getOrigin());
      }
    }, 600);
  }
  animate(dragX, PACK_WIDTH, dbg.scaleT({ duration: 0.3 }));
  animate(curlOpacity, 0, dbg.scaleT({ duration: 0.3 })).then(async () => {
    await dbg.gate("tear");
    onTear?.();
  });
}

State Changes

  • pack.dragX 0 → PACK_WIDTH (192)
  • pack.curlOpacity 1 → 0
  • pack.pendingOpen false → true (if autoOpenOnTear)
Feeds next: If autoOpenOnTear is set, arms a 600ms fallback timer and sets pendingOpen which triggers onLayoutAnimationComplete to continue to step 4 (elevate-card) via proceedToOpen(). Otherwise, the tear just unseals; the user must click or drag to proceed.

function handleCardPointerUp() {
  if (!isCardDragging.current || hasCardTriggered.current) return;
  isCardDragging.current = false;
  const current = cardDragY.get();

  if (current >= DRAG_UP_THRESHOLD) {
    hasCardTriggered.current = true;
    animate(cardDragY, 180, dbg.scaleT({
      type: "spring", stiffness: 300, damping: 30
    })).then(async () => {
      await dbg.gate("card-lift");
      await proceedToOpen();
    });
  } else {
    animate(cardDragY, 0, dbg.scaleT({
      type: "spring", stiffness: 400, damping: 25
    }));
  }
}

State Changes

  • pack.cardDragY user drag value → 180
Feeds next: On gate release, calls proceedToOpen() which fires step 4 (elevate-card) then step 5 (request-open). Alternative path to step 3.

useLayoutEffect(() => {
  if (!isClosing || collapseState) return;

  dbgRef.current.mark("collapse-measure");

  // ... find visible top-left card via bounding rects ...

  const targetRect = cards[targetIndex].getBoundingClientRect();
  const offsets = cards.map((card, i) => {
    if (i === targetIndex) return { x: 0, y: 0 };
    const rect = card.getBoundingClientRect();
    return {
      x: targetRect.left - rect.left,
      y: targetRect.top - rect.top,
    };
  });

  collapseCountRef.current = 0;
  collapseCompleteFiredRef.current = false;
  expectedCollapseRef.current = skills.length - 1;
  setCollapseState({ targetIndex, offsets });

  dbgRef.current.mark("collapse-fan");

  // Card 0 collapse — driven imperatively
  if (targetIndex !== 0) {
    animate(card0X, offsets[0].x, springConfig);
    animate(card0Y, offsets[0].y, springConfig).then(() => {
      dispatchCollapseComplete();
    });
  }
}, [isClosing, collapseState, ...]);

State Changes

  • drawer.collapseState null → { targetIndex, offsets }
  • drawer.targetIndex null → visible top-left card index
  • drawer.collapseCount 0 → 0 (reset)
  • drawer.expectedCollapseCount 0 → skills.length - 1
  • drawer.collapseCompleteFiredRef false → false (reset)
Feeds next: Immediately marks collapse-fan (step 11) in the same useLayoutEffect. Setting collapseState triggers a re-render where each non-target card gets its collapse animation.
