What happens in the session

1. Resolve context

Reads repo state, source artifacts, and relevant prior AFPS outputs before asking deeper questions.

2. Work the frame

Runs an interview, research synthesis, planning loop, build loop, audit, or shipping pass based on its type.

3. Gate the result

Uses review artifacts, checks, or approval language before writing canonical outputs or routing onward.
